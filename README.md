# Global-Airbnb-Performance-Dashboard
📊 Project Overview

This project is an interactive Power BI dashboard built to analyze Airbnb listing performance across multiple global cities.

Using Airbnb listing data, I transformed hundreds of thousands of raw records into meaningful business insights through data cleaning, modeling, and visualization.

The dashboard enables users to explore listing trends, property distribution, market share, and customer ratings, helping understand how Airbnb performs across different locations.

This project demonstrates practical skills in Power BI, data transformation, DAX calculations, and interactive dashboard design.

Unit tested all measures and KPI's using sql server.

🚀 Key Features

✔ Transformed 270K+ Airbnb listing records into structured analytical data
✔ Built interactive dashboards with drill-down analysis
✔ Implemented DAX measures for key performance metrics
✔ Used Power Query for data cleaning and transformation
✔ Implemented bookmarks and buttons for smooth navigation
✔ Created business-focused visualizations for decision making

📈 Dashboard Pages
1️⃣ Overview Dashboard

Provides a high-level summary of Airbnb performance.

Key metrics include:

Total Listings

Total Cities

Total Hosts

Property Types

Total Reviews

Visualizations include:

Listing growth trends over time

Distribution of room types

Superhost vs non-superhost market share

🚀>**Business Questions Answered**

Which cities have the highest number of Airbnb listings?

What property types dominate the market?

How do customer ratings vary across cities?

What is the distribution of superhost vs non-superhost listings?

⭐ Ratings Analysis Dashboard

This page focuses on analyzing customer satisfaction and review scores.

Insights include:

Average ratings by city

Cleanliness score

Communication score

Location score

Accuracy score

This helps identify cities with the highest guest satisfaction levels.

📊 Key Insights

Entire homes/apartments dominate the Airbnb property market.

Certain cities contribute significantly to total listings.

Customer ratings across most cities remain consistently high.

Listing growth increased rapidly between 2012 and 2016 before stabilizing.
